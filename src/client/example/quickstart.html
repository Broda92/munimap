Krok za krokem, jak vložit na stránku základní mapku

Krok 1 - nahrání knihovny

Pro vložení mapy s budovami a místnostmi Masarykovy univerzity na webovou stránku je nejprve potřeba nahrát knihovnu munimap. To uděláte tak, že do hlavičky stránky (mezi značky <head> a </head>) vložíte následující kód: 

  <script src="//{jpad.prodDomain}{jpad.appPath}munimaplib.js" type="text/javascript"></script>

Krok 2 - umístění mapy

Pak na libovolné místo Vaší webové stránky (mezi značky <body> a </body>) vložte značku <div> s atributem id nastaveným například na hodnotu map, na tomto místě se pak zobrazí mapa. Id elementu může být libovolné, stejné id ale musíte předat funkci munimap.create. Velikost mapy ovlivníte nastavením CSS stylu pro element <div> (vlastnosti width a height). 

  <div id="map">
  </div>

Krok 3 - vytvoření a nastavení mapy

Mapu vytvoříte zavoláním funkce munimap.create. Na konec stránky (před uzavírací značku </body>) vložte následující kód: 

  <script type="text/javascript">
    munimap.create({
      target: 'map'
    });
  </script>

V příkladu použité parametry funkce munimap.create jsou následující:

  • target - určuje id elementu, do kterého se mapa vykreslí

Pomocí parametrů funkce munimap.create můžete ovlivnit další nastavení mapy, jako je zvýraznění budov, změna jazyka či podkladové mapy. Více se dozvíte v naší dokumentaci a příkladech použití.

Výsledek

Jestliže jste postupovali podle předchozího návodu, uvidíte na své stránce následující interaktivní mapu:

Vyzkoušejte si sami

Následující kód jednoduché HTML stránky si můžete zkopírovat a uložit do souboru s příponou .html. Dostanete tak prázdnou stránku se základní mapou. 

<html>
  <head>
    <title>Jednoduchá mapka</title>
    <meta charset="UTF-8" />
    <script src="//{jpad.prodDomain}{jpad.appPath}munimaplib.js" type="text/javascript"></script>
    <style>
      #map { width: 800px; height: 400px; }
    </style>
  </head>
  <body>
    <div id="map">
    </div>
    <script type="text/javascript">
      munimap.create({
	target: 'map'
      });
    </script>
  </body>
</html>
