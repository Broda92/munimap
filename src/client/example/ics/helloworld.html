munimap - vložení mapky na stránku

Do hlavičky stránky (mezi značky <head> a </head>) vložte následující kód, který zajistí nahrání knihovny munimap do Vaší stránky: 

  <script src="//maps.muni.cz/munimap/ics/map/lib/lib.js" type="text/javascript"></script>

Na libovolné místo Vaší webové stráky (mezi značky <body> a </body>) vložte značku <div> s parametrem id nastaveným na hodnotu map, na tomto místě se pak zobrazí mapka. Velikost mapky ovlivníte nastavením CSS elementu <div> (vlastnosti width a height). 

  <div id="map">
  </div>

Na konec stránky (před uzavírací značku </body>) vložte následující kód, ten ovládá nastavení mapy: 

  <script type="text/javascript">
    ics.map.create({
      target: 'map'
    });
  </script>

Výsledný kód jednoduché HTML stránky by mohl vypadat následovně: 

<html>
  <head>
    <title>Jednoduchá mapka</title>
    <meta charset="UTF-8" />
    <script src="//maps.muni.cz/munimap/ics/map/lib/lib.js" type="text/javascript"></script>
    <style>
      #map { width: 800px; height: 400px; }
    </style>
  </head>
  <body>
    <div id="map">
    </div>
    <script type="text/javascript">
      ics.map.create({
	target: 'map'
      });
    </script>
  </body>
</html>
