munimap - reset mapy

Pomocí metody ics.map.reset lze přenastavit parametry mapy. Chybějící parametry se nastaví na výchozí hodnotu.

- Tlačítko pro reset mapy (id je resetButton) 
<script type="text/javascript">
  ics.map.create({
    target: 'map',
    zoom: 17,
    markers: ['BMA01']
  }).then(function(map) {
    var button = document.getElementById('resetButton');
    button.onclick = function() {
      ics.map.reset(map, {
        markers: ['BMB01']
      });
    };
  });
</script>
