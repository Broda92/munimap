munimap - reset mapy

Pomocí funkce ics.map.reset lze přenastavit parametry mapy. Chybějící parametry (oproti parametrům původního volání funkce ics.map.create) se nastaví na výchozí hodnotu.

Funkci ics.map.reset voláme v callbacku funkce ics.map.create, kde již máme k dispozici objekt mapy. Tento objekt předáme jako parametr funkci ics.map.reset.

- Tlačítko pro reset mapy (id je resetButton) 
<script type="text/javascript">
  ics.map.create({
    target: 'map',
    zoom: 17,
    markers: ['BMA01']
  }).then(function(map) {
    var button = document.getElementById('resetButton');
    button.onclick = function() {
      ics.map.reset(map, {
        markers: ['BMB01']
      });
    };
  });
</script>
