munimap - vložení mapy na stránku

munimap > příklady použití > vložení mapy na stránku

Pro vložení mapy s budovami a místnostmi Masarykovy univerzity na webovou stránku je třeba nahrát knihovnu munimap a zavolat na stránce metodu munimap.create. Do hlavičky stránky (mezi značky <head> a </head>) vložte následující kód, který zajistí nahrání knihovny munimap do Vaší stránky: 

  <script src="//{jpad.prodDomain}{jpad.appPath}munimaplib.js" type="text/javascript"></script>

Na libovolné místo Vaší webové stránky (mezi značky <body> a </body>) vložte značku <div> s atributem id nastaveným například na hodnotu map, na tomto místě se pak zobrazí mapa. Id elementu může být libovolné, stejné id ale musíte předat metodě munimap.create. Velikost mapy ovlivníte nastavením CSS stylu pro element <div> (vlastnosti width a height). 

  <div id="map">
  </div>

Na konec stránky (před uzavírací značku </body>) vložte následující kód, ten ovládá nastavení mapy: 

  <script type="text/javascript">
    munimap.create({
      target: 'map'
    });
  </script>

Výsledný kód jednoduché HTML stránky by mohl vypadat následovně: 

<html>
  <head>
    <title>Jednoduchá mapka</title>
    <meta charset="UTF-8" />
    <script src="//{jpad.prodDomain}{jpad.appPath}munimaplib.js" type="text/javascript"></script>
    <style>
      #map { width: 800px; height: 400px; }
    </style>
  </head>
  <body>
    <div id="map">
    </div>
    <script type="text/javascript">
      munimap.create({
	target: 'map'
      });
    </script>
  </body>
</html>

V příkladu použité parametry funkce munimap.create jsou následující:

  • target - určuje id elementu, do kterého se mapa vykreslí
